# Demo runbook + pitch (~2-3 min)

**One-liner:** "Getting 65,000 fans to Hard Rock safely is solved on paper and a disaster in
practice. Kickoff routes each fan to the right arrival method for their match — in their language."

## Run of show
1. **Stakes (0:00):** the 2024 Copa América crush at THIS venue. Real, citable. One line, let it land.
2. **Core live (0:30):** "Brazil fan, staying in Brickell, June 24" -> ranked plan, **leave by 4:10 PM**, parking hard stop. Tap a source.
3. **Map (1:15):** Wynwood/Brickell hot zones + watch parties; tap one for its playlist.
4. **Wow (1:45):** speak the query in **Portuguese** -> the plan reads back.
5. **"Is this real?" (2:10):** pull up the FIFA Miami page live. Most teams fold here. You won't.
6. **Close (2:30):** 30-sec Cursor time-lapse — Agent mode building this, work split across background agents.
